Variable substitution
===================================

If the property is not declared in the configuration file, logback will look for it in the System properties. 
Java system properties can be set on the command line as shown next:

java -DUSER_HOME="/home/sebastien" MyApp2

Example:

<configuration>

  <appender name="FILE" class="ch.qos.logback.core.FileAppender">
    <file>${USER_HOME}/myApp.log</file>
    <encoder>
      <pattern>%msg%n</pattern>
    </encoder>
  </appender>

  <root level="debug">
    <appender-ref ref="FILE" />
  </root>
</configuration>


When multiple variables are needed, it may be more convenient to create a separate properties file.


<configuration>

  <property file="src/main/java/chapters/configuration/variables1.properties" />

  <appender name="FILE" class="ch.qos.logback.core.FileAppender">
     <file>${USER_HOME}/myApp.log</file>
     <encoder>
       <pattern>%msg%n</pattern>
     </encoder>
   </appender>

   <root level="debug">
     <appender-ref ref="FILE" />
   </root>
</configuration>


variables1.properties
USER_HOME=/home/sebastien

Default values for variables can be specified using the ":-" operator. 

For example, assuming the variable named aName is not defined, "${aName:-golden}" will be interpreted as "golden".


Real example
===================================

<!DOCTYPE xml>
<configuration>

  <property file="${ESCHOLAR_HOME}uid/conf/uid.properties"/>

  <appender name="STDOUT" class="ch.qos.logback.core.ConsoleAppender">
    <encoder>
      <pattern>
        %d{HH:mm:ss} [%thread] %-5level %class.%method\(line:%line\) -  - %msg%n
      </pattern>
    </encoder>
  </appender>
  
  <logger name="org.apache.activemq.transport" level="WARN"/>
  <logger name="org.apache.http" level="ERROR" />
  <logger name="org.apache.wire" level="ERROR" />
  <logger name="httpclient" level="ERROR" />
  <Logger name="software.amazon.awssdk" level="WARN" />
  <Logger name="software.amazon.awssdk.request" level="DEBUG" />
  <Logger name="org.apache.ibatis.transaction.jdbc.JdbcTransaction" level="WARN" />
  <Logger name="org.apache.ibatis.datasource.pooled.PooledDataSource" level="WARN" />
  
  
  
  <root level="${uid.api.logLevel:-INFO}">
    <appender-ref ref="STDOUT"/>
  </root>

</configuration>

uid.driverName=escholar.jdbc.sqlserver.SQLServerDriver
